# Sacrifice Sanctuary: Giddy Up-Despair

![](/game/gui/main_menu.png)

A death game visual novel, built using Ren'py.

----
## Development setup

Requirements: renpy, python3, pip

1. Clone repo - `git clone https://github.com/MillieMD/sacrifice-sanctuary`
1. Run `pip install -r requirements.txt -t ./game/python-packages` to install python dependencies
